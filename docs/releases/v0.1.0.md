# v0.1.0 发布说明

这是 `warp-socks` 的初版发布。

## 版本定位

当前版本提供一个轻量、单用途、低副作用的 WARP SOCKS5 出口，支持三种注册后端：

- `teams`
- `wgcf-free`
- `wgcf-plus`

## 本版内容

- 项目默认命名统一为 `warp-socks`
- 镜像新增极简 `HEALTHCHECK`
- 文档拆分为首页入口加专题文档
- 安全边界中明确强调无认证 SOCKS5 的对外暴露风险
- `microsocks` 构建源固定到上游已发布 tag `v1.0.5`

## 默认安全边界

默认端口映射保持为 `127.0.0.1:1080 -> 1080`。

如果你把 `HOST_BIND_IP=0.0.0.0`，就会直接暴露无认证 SOCKS5 代理。除非你已有额外 ACL / 防火墙限制，否则不建议这样做。

## 验证摘要

本版发布前已完成以下验证：

- `docker compose config -q`
- `docker compose build`
- `docker compose up -d`
- `docker compose ps` 显示 `healthy`
- 宿主机经 `curl --socks5 127.0.0.1:1080 https://cloudflare.com/cdn-cgi/trace` 返回 `warp=on`
